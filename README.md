## 技术栈

| 层级 | 技术 |
|------|------|
| 桌面壳 | Electron 28 |
| 前端框架 | Vue 3 (Composition API) |
| 类型系统 | TypeScript 5 |
| 构建工具 | electron-vite + Vite 5 |
| 状态管理 | Pinia |
| 路由 | Vue Router 4 |
| 样式 | LESS |
| 压缩包解析 | adm-zip（支持 ZIP / CBZ） |
| 数据持久化 | electron-store |

## 项目结构

```
manga-box-pc/
├── electron.vite.config.ts     # 构建配置
├── tsconfig.json               # TS 根配置（references 模式）
├── tsconfig.node.json          # 主进程 / 预加载 TS 配置
├── tsconfig.web.json           # 渲染进程 TS 配置
└── src/
    ├── main/
    │   └── index.ts            # Electron 主进程（窗口、IPC、文件系统）
    ├── preload/
    │   └── index.ts            # 预加载脚本（向渲染进程暴露 electronAPI）
    └── renderer/
        ├── index.html
        └── src/
            ├── env.d.ts        # 全局类型声明（Book、ReadProgress、ElectronAPI）
            ├── main.ts         # Vue 应用入口
            ├── router/         # 页面路由
            ├── stores/
            │   └── library.ts  # Pinia Store（书库、阅读进度）
            ├── styles/
            │   ├── variables.less  # LESS 变量 & Mixin
            │   └── global.less     # 全局基础样式
            ├── components/
            │   ├── TitleBar.vue    # 自定义无边框标题栏
            │   ├── Sidebar.vue     # 左侧导航栏
            │   ├── BookCard.vue    # 漫画卡片（网格视图）
            │   └── SettingsPanel.vue # 设置面板
            └── views/
                ├── HomeView.vue    # 书架主页
                └── ReaderView.vue  # 漫画阅读器
```

## 开发环境

### 前置要求

- Node.js >= 18
- npm >= 9

```bash
npm install
```


## 使用说明

### 添加漫画

MangaBox 支持三种方式导入本地漫画：

#### 方式一：设置书库文件夹（推荐）

适合漫画数量较多、统一存放的情况。

1. 点击左侧导航栏 **「书库路径」**，或点击空书架页的 **「设置书库路径」** 按钮。
2. 在弹出的系统文件夹选择器中，选择存放漫画的根目录。
3. 程序会自动扫描该目录下的所有：
   - **子文件夹**（文件夹内需包含图片文件）
   - **压缩包文件**（`.zip` / `.cbz` / `.cbr`）
4. 扫描完成后书架自动更新。

> 书库文件夹内的目录结构示例：
> ```
> 我的漫画/
> ├── 进击的巨人/
> │   ├── 001.jpg
> │   ├── 002.jpg
> │   └── ...
> ├── 鬼灭之刃.zip
> └── 咒术回战.cbz
> ```

#### 方式二：单本添加

1. 点击左侧导航栏 **「添加漫画」**。
2. 在弹出的对话框中选择单个压缩包文件（支持多选）或文件夹。
3. 所选漫画会直接添加到书架，无需设置书库路径。

#### 方式三：拖拽导入

直接将漫画**文件夹**或**压缩包文件**拖拽到程序窗口，松开鼠标即可自动导入。

### 书架页面

书架是软件的主页面，提供以下功能：

#### 视图切换

- **全部漫画**：展示书架中所有漫画。
- **最近阅读**：按最后阅读时间倒序排列，最多显示 12 本。

#### 搜索

在右上角的搜索框中输入关键词，实时过滤当前视图中的漫画标题。

#### 网格 / 列表视图

点击右上角的网格图标或列表图标切换显示方式：

- **网格视图**：显示封面缩略图、页数，悬停后出现「开始阅读 / 继续阅读」按钮。
- **列表视图**：显示封面、标题、页数、阅读进度条，适合漫画较多时快速浏览。

#### 阅读进度

已读过的漫画会在封面底部显示**紫色进度条**，列表视图中还会显示当前页码与总页数。

#### 右键菜单

在网格视图中**右键点击**漫画卡片，可以：

- **打开阅读**：进入阅读器。
- **从书架移除**：从书架删除该条目（不会删除本地文件）。

#### 刷新

点击右上角 **「刷新」** 按钮，重新扫描书库文件夹，同步新增或删除的漫画。

---

### 阅读器

点击任意漫画卡片（或右键菜单中的「打开阅读」）进入阅读器。

#### 阅读模式

顶部工具栏中可切换三种阅读模式：

| 模式 | 说明 |
|------|------|
| **单页** | 每次显示一张图片，适合大多数普通漫画 |
| **双页** | 同时显示相邻两页，适合扫描版或宽幅漫画 |
| **长条** | 图片纵向连续滚动，适合条漫 / 韩漫 |

#### 翻页

- **单页 / 双页模式**：
  - 点击页面**左侧区域**：上一页
  - 点击页面**右侧区域**：下一页
  - 滚动**鼠标滚轮**：上下翻页
- **长条模式**：直接上下滚动页面

#### 缩放（单页 / 双页模式）

- 点击顶部工具栏的 **`−`** / **`+`** 按钮调整缩放比例（范围 30% ~ 300%）
- 按 **重置** 按钮恢复为 100%
- `Ctrl + 滚轮` 也可缩放

#### 进度条

底部进度条显示当前阅读位置：

- 点击进度条上的任意位置可**跳转**到对应页面
- 悬停在进度条上会显示目标页码提示

#### 界面隐藏

- 点击顶部工具栏右侧的**眼睛图标**，或按 `H` 键，可隐藏顶部栏和底部进度条，进入沉浸阅读模式。
- 鼠标移动或点击任意位置可恢复界面显示（3 秒无操作后自动隐藏）。

#### 全屏

点击工具栏的**全屏图标**或按 `F` 键进入全屏模式，按 `Esc` 退出。

#### 进度自动保存

退出阅读器时会自动保存当前页码，下次打开同一本漫画时从上次位置继续。

---

### 键盘快捷键

在阅读器页面内，以下快捷键有效：

| 按键 | 功能 |
|------|------|
| `←` / `A` | 上一页（单页 / 双页模式） |
| `→` / `D` | 下一页（单页 / 双页模式） |
| `↑` / `W` | 向上滚动（长条模式） |
| `↓` / `S` | 向下滚动（长条模式） |
| `+` / `=` | 放大 |
| `-` | 缩小 |
| `0` | 重置缩放为 100% |
| `H` | 隐藏 / 显示顶部与底部工具栏 |
| `F` | 进入 / 退出全屏 |
| `Esc` | 退出全屏 或 返回书架 |
| `?` | 显示 / 隐藏快捷键帮助面板 |

---

### 设置

点击左侧导航栏底部的**齿轮图标**打开设置面板，可配置：

| 选项 | 说明 |
|------|------|
| 书库路径 | 查看或更改当前书库文件夹路径 |
| 默认阅读模式 | 打开漫画时默认使用的模式（单页 / 双页 / 长条） |
| 阅读方向 | 从左到右（普通漫画）或从右到左（日式漫画） |

所有设置自动保存，下次启动时生效。

## 打包发布

### Windows

```bash
npm run build:win
```

输出 `.exe` 安装包至 `dist/` 目录。

### macOS

```bash
npm run build:mac
```

输出 `.dmg` 安装包至 `dist/` 目录。

### Linux

```bash
npm run build:linux
```

输出 `.AppImage` 等格式至 `dist/` 目录。
